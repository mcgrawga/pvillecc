Administration Instructions

{% endblock %} {% block content %} Here is a list of stuff you can do on your website. Click the link that describes what you want to do and read the instructions. If you want to do something, and do not see that something described, use the contact information and we will help. Also, all of these instructions assume you are within the admin console of your club's website (www.yourclub.com/admin)

Before we get to the section about what you can do, let's make sure we cover the things you have to do in order for the handicap system to function properly. The following items must be completed before the start of every golf season.
  • Define the upcoming season by entering the season start and end dates.
  • Define the upcoming season by entering the handicap revision dates.

Change the text of a page.
  1. Click on the Flatpages link.
  2. Click on the page you would like to change.
  3. Make the text changes to the title or content, do not change the URL or SITE.
  4. Click the save button. The changes should now show up on the main site.

Define the upcoming season by entering the start and end dates.
  1. Click the add link to the right of the Seasons link on the main page
  2. Enter the appropriate information, bold items are required.
  3. Click the save button. The season has been added to the system. To check click the Seasons link on the main page and make sure the season you added is displayed in the list of seasons.


Define the upcoming season by entering the handicap revision dates.
  1. For each handicap revision date in the upcoming season click the add link to the right of the Handicap revision dates link on the main page. For a list of handicap revision dates visit www.iowagolf.org
  2. Enter the appropriate information, bold items are required.
  3. Click the save button. The handicap revision date has been added to the system. To check, click the Handicap revision dates link on the main page and make sure the revision date you added is displayed in the list of handicap revision dates.


Enter a score for a golfer.
  1. Click Enter Golfer Scores on the main page
  2. Click the login button to the right of the golfer you want to enter a score for.
  3. Click quick score next to the appropriate golf course.
  4. Enter the score.
  5. Click save score.


Peer review golf scores.
  1. Click Peer/Score Review on the main page.
  2. Click the view link to the right of the golfer whose scores you would like to review or choose a time period for which to review all golfers scores.


Print a 20 score history for every member.
  1. Click Print Score History on the main page.
  2. You will be prompted to open a *.csv file with the default application on your local computer, most likely excel.
  3. The opened file will contain a 20 score history for every member.


Print a handicap history for every member.
  1. Click Print Handicap History on the main page.
  2. You will be prompted to open a *.csv file with the default application on your local computer, most likely excel.
  3. The opened file will contain the handicap history for every member.


Contact support.
  1. Click Contact Support on the main page.
  2. Use the information on the page to contact support, i.e. email, phone, address, etc...
